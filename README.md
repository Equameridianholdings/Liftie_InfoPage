# Liftie Carpool Info Page

A modern, responsive information page for the Liftie carpooling application, inspired by professional ride-sharing services.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Modern UI**: Clean, professional design with smooth animations and transitions
- **Key Sections**:
  - Header with navigation menu
  - Hero section with video placeholder and call-to-action
  - Main content highlighting benefits
  - Mission statement banner
  - About section explaining the service
  - Features grid showcasing 6 key benefits
  - Final call-to-action section
  - Comprehensive footer

## How to Use

1. Open `index.html` in any modern web browser
2. Customize the content to match your specific needs
3. Replace the placeholder image URL with your own images
4. Update colors in `styles.css` if needed (primary color: `#5a9e6f`)

## Customization

### Colors
The main brand color is green (`#5a9e6f`). To change it, search and replace in `styles.css`:
- Primary green: `#5a9e6f`
- Hover state: `#4a8e5f`
- Dark background: `#2c3e50`

### Content
Edit the text content directly in `index.html`:
- Hero section text and CTAs
- Feature descriptions
- Mission statement
- About section details

### Images
Replace the Unsplash placeholder URL in the hero section with your own carpooling images or videos.
